# 字符图生成器 — 256 色字符画

将任意图片转换为彩色字符画。支持自定义填充字符（ASCII / 方块 / 中文）、背景颜色、导出为 PNG 图片 / HTML / TXT。

## 快速开始

### 方式一：直接运行 EXE

下载 `dist/字符图生成器.exe`，双击启动。选择图片 → 点击「生成预览」即可。

### 方式二：Python 源码运行

```bash
pip install pillow
python Python字符图生成程序编写.py
```

## 功能特性

- **256 色渲染** — 每个字符保留原图的颜色信息
- **自定义字符** — 支持任意字符序列（暗→亮），包括中文、Unicode 符号
- **内置预设** — ASCII 经典 / 方块字符 █▓▒░ / 单字符模式
- **背景颜色** — 预设（黑/白/灰）+ RGB 数值 + 系统调色板，实时预览
- **导出格式** — PNG 图片 / HTML 网页 / TXT 纯文本
- **缩放** — Ctrl + 滚轮 放大/缩小预览

## 参数说明

| 参数 | 默认值 | 说明 |
|---|---|---|
| 宽度 | 160 | 每行字符数（30~500） |
| 宽高比 | 0.50 | 补偿终端/字体字符宽高比 |
| 字符 ramp | ASCII 经典 | 暗→亮排列的字符序列 |
| 背景色 | #1e1e1e | 预览背景 / 导出图片背景 |

### 字符 ramp 原理

字符按**视觉密度**从暗到亮排列。亮度越低的像素映射到越靠前的字符：

```
暗 ──────────────────────────────→ 亮
$@B%8&WM#*...zcvrjft...i!lI;:,.'
█▓▒░                     (方块)
龙龘鑫森淼焱垚...           (中文)
```

- **多字符模式**：不同亮度用不同字符 → 丰富层次感
- **单字符模式**（如 `--chars "█"`）：同一字符不同颜色 → 马赛克色块画

## 导出格式

| 格式 | 说明 | 适用场景 |
|---|---|---|
| **PNG** | 逐字符渲染，目标 ~1280×720px | 社交分享、壁纸 |
| **HTML** | 浏览器查看，保留颜色 | 网页嵌入、存档 |
| **TXT** | 纯文本 | 终端 cat、粘贴到聊天 |

## 字符预设示例

| 预设 | 效果 |
|---|---|
| ASCII 经典 | `$@B%8&WM#*...` 层次丰富 |
| 方块字符 | `█▓▒░ ` 像素画风格 |
| 单字符 █ | 纯色块，保留下原图颜色 |
| 单字符 我 | 全部用"我"字，保留下原图颜色 |
